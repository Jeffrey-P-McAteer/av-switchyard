av-switchyard

av-switchyard is a primarially command-line Go utility which performs the following capabilities

  • av-switchyard scan
    • For every network interface, report all accessible machines and AV equipment.
  • av-switchyard
    • With no arguments, or the optional explicit form av-switchyard daemon, runs a server which reads the file av-switchyard.toml to connect ArtNet gear to grandMA3.
  • av-switchyard update [--upgrade-version=va.b.c]
    • Self-upgrade capability. Replaces the currently-running binary with the specified (or most recent) version downloaded from the releases page at av-switchyard/releases
